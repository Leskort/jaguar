# Videos

Place your video files here.

## Hero Section Video

- File name: `hero-video.mp4` (required)
- Optional: `hero-video.webm` (for better browser compatibility)
- Recommended size: 5-10 MB
- Resolution: 1920x1080 or 1280x720
- Duration: 10-30 seconds (will be looped)
- Format: MP4 (H.264 codec) for best compatibility

## Usage

The hero video is automatically used on the homepage. Make sure to place the video file in this directory with the name `hero-video.mp4`.
