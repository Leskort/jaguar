# Team Photos

Place team member photos in this directory.

## Recommended file names:
- `jenya-uk.jpg` - for Jenya (UK Branch)
- `jenya-uk-ua.jpg` - for Jenya (UK, UA)
- `chief-electrician.jpg` - for Chief Electrician
- `serhiy-ua.jpg` - for Serhiy (UA)
- `ihor-ua.jpg` - for Ihor (UA)

## Image specifications:
- Format: JPG, PNG, or WebP
- Recommended size: 400x400px or larger (square aspect ratio)
- Recommended resolution: 300 DPI or higher
- File size: Keep under 500KB for optimal loading

## Tips:
- Use square images (1:1 aspect ratio) for best results
- Optimize images before uploading
- Use professional photos or automotive-themed stock images
- Images will be displayed in circular frames

## Example sources for automotive-themed images:
- Unsplash (search: "mechanic", "automotive technician", "car service")
- Pexels (search: "auto mechanic", "car repair")
- Use professional team photos if available
